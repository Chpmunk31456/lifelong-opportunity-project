# Governance

The Lifelong Opportunity Guides project is a volunteer-led public education project created and directed by Alberto “Al” Leiva.

## Decision Principles

Project decisions should prioritize:

1. Factual accuracy and transparent uncertainty
2. Accessibility and plain language
3. Safety, legality, privacy, and ethical AI use
4. Free and low-cost learning pathways
5. Protection against guarantees and predatory programs
6. Respectful localization and human review
7. Long-term public availability

## Maintainer Responsibilities

Maintainers should:

- Protect the license and attribution
- Preserve revision history
- Review material corrections against authoritative sources
- Document factual-review dates
- Separate drafts from reviewed editions
- Avoid unsupported employment or education claims
- Respond respectfully to accessibility and translation concerns

## Review and Release

A material revision should identify:

- Guide and language
- Previous and new version
- Factual-review date
- Sources that changed
- Accessibility review status
- Human translation-review status, when applicable

Minor formatting corrections may retain the existing version when they do not change meaning. Factual or structural changes should receive a documented version update.

## Succession

To help the project continue beyond its founder:

- At least one trusted co-maintainer should eventually have repository access.
- Governance, release, and factual-review procedures should remain documented.
- Archival releases should be deposited in stable public repositories.
- Future maintainers must preserve attribution, noncommercial licensing, accessibility, and ethical safeguards.
- If active maintenance ends, the repository should remain publicly readable and clearly identify the last factual-review date.
